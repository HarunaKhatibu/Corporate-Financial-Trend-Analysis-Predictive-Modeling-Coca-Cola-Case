# Corporate-Financial-Trend-Analysis-Predictive-Modeling-Coca-Cola-Case
This project presents an end-to-end data science workflow focused on financial forecasting for Coca-Cola. The objective is to transform raw, unstructured financial data into a clean analytical dataset and build machine learning models to predict revenue and net income trends for the next ten years. 
The project demonstrates practical skills in data cleaning, financial data transformation, exploratory data analysis, regression-based time series modeling, and long-term business forecasting.
### Key Highlights
  - Performed advanced data cleaning on unstructured Excel financial data
  - Standardized currency formats and handled negative values and inconsistencies
  - Conducted exploratory trend analysis on revenue and net income
  - Built regression-based forecasting models for 10-year financial prediction
  - Generated future projections to support strategic business insights
### Technologies Used
  - Python
  - Pandas
  - NumPY
  - Scikit-learn
  - Matplotlib / Seaborn
  - Jupyter Notebook
